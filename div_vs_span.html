<div> element:

  • Used to define a division or a section of the document.
  • Group elements for styling purposes, by assigning an id or class attribute to the div element and then apply styles which will affect all elements n the div container.

Examples:

Week 1

This week, you will be learning about...week 1 stuff

  1. Line of code
  2. Line of code

Week 2

This week, you will be learning about...week 2 stuff

  1. Line of code
  2. Line of code

<span> element:

  • To add styling to part of a sentence (inline).
  • Manipulate part of a sentence using JavaScript.
  • When no other HTML element is applicable, you can use <span> (and <div>) to add attributes such as class and id.
  • <div> is a block level element(for a block of space) while <span> is an inline element (for within a line or phrase.
